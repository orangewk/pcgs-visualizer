# Complex Gaussian Splat Cloud 仮説 — コンパクトノート

## 0. この文書の位置づけ

これは、量子・観測・隠れ次元を **Gaussian splat / rendering** の語彙で再構成するための概念ノートである。  
現時点では「新しい物理法則」や「完成した万物理論」ではなく、既存の量子論・余剰次元・観測問題を、CG 的な表現モデルとして読み替える試みである。

主張を短く言えば、次のようになる。

> 世界の基礎を「点粒子」や「古典的な波」として見るのではなく、  
> 高次元状態空間上の **complex anisotropic Gaussian splat cloud** として表現する。  
> 観測される現実は、その cloud を隠れ次元方向に周辺化し、有限分解能でレンダーし、個々の観測では確率的に sample した結果である。

---

## 1. 出発点

出発点は素朴である。

現実は、ミクロには点ではなく、広がり・揺らぎ・干渉を持つものとして現れる。  
一方、マクロには境界を持つ物体、連続的な面、安定した blob の集合として見える。

CG の Gaussian splatting では、シーンを多数の anisotropic Gaussian primitive の集合として表現し、それをカメラ方向へ投影・合成して画像にする。  
この構造は、量子状態を「広がった複素振幅」として表し、観測時に確率分布を得る構造とよく似ている。

したがって、仮説の出発点は次である。

> 見えている世界が局所的 blob / splat の集合として現れるなら、  
> 世界記述の基本 primitive を Gaussian splat として置いてみる。

ただし、ここでの splat は普通の密度雲ではない。  
量子論に合わせるため、**複素振幅・位相・運動量・干渉能力**を持つ splat とする。

## 2. 基本定式化

見える3次元座標を \(r\)、観測できない余剰次元座標を \(y\) とする。

\[
z=(r,y)
\]

10次元時空を念頭に置く場合、通常は

\[
3\text{ visible space} + 6\text{ hidden space} + 1\text{ time}
\]

と考え、空間部分を

\[
M=\mathbb{R}^3\times K_6
\]

と置く。ここで \(K_6\) はコンパクトな隠れ次元空間である。

隠れ次元のサイズは、仮にプランク長スケールとする。

\[
\mathrm{size}(K_6)\sim \ell_P
\]

これは「そうである」と実証された値ではなく、見えている現実と矛盾しにくい保守的な隠し方としての仮定である。

## 3. 状態：complex Gaussian splat cloud

量子状態を、複素 Gaussian primitive の重ね合わせとして書く。

\[
\Psi(r,y,t)=\sum_{j=1}^{K} c_j(t)g_j(r,y,t)\chi_j
\]

ここで、

- \(K\)：splat 数
- \(c_j\)：複素重み
- \(g_j\)：第 \(j\) Gaussian primitive
- \(\chi_j\)：内部状態。スピン、偏光、電荷、色荷など
- \(\Psi\)：全体の量子状態

1個の splat は次の形を持つ。

\[
g_j(z,t)=N_j\exp\left[
-\frac12(z-q_j)^TA_j(z-q_j)
+\frac{i}{\hbar}p_j^T(z-q_j)
+i\phi_j
\right]
\]

各 splat のパラメータは、

\[
(q_j,\ p_j,\ A_j,\ c_j,\ \phi_j,\ \chi_j)
\]

である。

| パラメータ | 意味 |
|---|---|
| \(q_j\) | 中心位置 |
| \(p_j\) | 運動量、位相勾配 |
| \(A_j\) | 広がり、向き、位相曲率 |
| \(c_j\) | 複素重み |
| \(\phi_j\) | 位相 |
| \(\chi_j\) | 内部量子数 |

重要なのは、splat を密度として足すのではない、という点である。

CG 的な密度合成なら、

\[
\rho(x)=\sum_j \alpha_jG_j(x)
\]

のように足す。  
しかし量子版では、先に複素振幅を足す。

\[
\Psi(x)=\sum_j c_jg_j(x)
\]

その後で、観測確率を絶対値二乗で得る。

\[
P(x)=|\Psi(x)|^2
\]

したがって、

\[
|\Psi|^2=
\sum_j |c_jg_j|^2+
\sum_{i\neq j}c_i^*c_jg_i^*g_j
\]

となり、第二項が干渉を生む。  
ここが、このモデルの量子性の中心である。

## 4. 観測：density render と event render

Gaussian splat renderer は、普通は一回ごとに単一結果を出す装置ではない。  
それは期待値としての画像、つまり density render を出す。

このモデルでも同じ区別を置く。

### 4.1 density render

量子状態から確率場を作る。

\[
P(x,t)=|\Psi(x,t)|^2
\]

これは「どこに観測されやすいか」の分布であり、一回の結果ではない。

### 4.2 event render

一回の観測では、確率場から1つの結果が sample される。

\[
x_{\mathrm{obs}}\sim P(x,t)
\]

つまり、

\[
\text{splat cloud}\rightarrow\text{probability field}\rightarrow\text{sampled event}
\]

である。

二重スリットで言えば、1個の光子はスクリーン上の1点として出る。  
しかし多数回繰り返すと、点のヒストグラムは安定した干渉縞に近づく。

\[
\hat P_N(x)\to P(x)
\]

揺れは概ね、

\[
\mathrm{noise}\sim \frac{1}{\sqrt N}
\]

で小さくなる。

したがって、ミクロな一回の観測は揺れるが、多数回・多数自由度・粗視化によってマクロな現実は安定して見える。

## 5. 隠れ次元の扱い

観測者が見えるのは \(r\) だけであり、\(y\) は直接見えない。  
したがって、見える確率分布は隠れ次元を積分消去して得る。

\[
P_{\mathrm{vis}}(r,t)=\int_{K_6}dy\,|\Psi(r,y,t)|^2
\]

隠れ次元が円のようにコンパクト化されているなら、

\[
y^a\sim y^a+2\pi R_a
\]

であり、仮に

\[
R_a\sim \ell_P
\]

と置く。

このとき、隠れ次元方向の非ゼロモードのエネルギーは概ね、

\[
E_n\sim \frac{\hbar c|n|}{R}
\]

となる。  
もし \(R\sim\ell_P\) なら、その励起はプランクエネルギー級になり、低エネルギーの観測ではほぼ現れない。

低エネルギーでは、

\[
\Psi(r,y,t)\approx \psi_0(r,t)\eta_0(y)
\]

となり、

\[
P_{\mathrm{vis}}(r,t)\approx |\psi_0(r,t)|^2
\]

が得られる。

つまり、隠れ次元は存在していても、通常の観測ではゼロモードだけが残り、見える世界は3次元の確率分布として現れる。

## 6. なぜ Gaussian なのか

Gaussian を採用する理由は、唯一絶対の証明ではない。  
しかし、表現 primitive としては複数の自然な理由がある。

### 6.1 有限分解能での最小表現

観測で直接わかるのは、多くの場合、中心・広がり・向き・強度である。  
平均と共分散だけを指定するとき、余計な構造を最小化した分布として Gaussian は自然である。

### 6.2 最小不確定波束

量子では、位置と運動量を同時に無限精度で決められない。

\[
\Delta x\Delta p\geq\frac{\hbar}{2}
\]

この下限と相性がよい代表的な波束が Gaussian である。

### 6.3 周辺化しても Gaussian 的形状が保たれる

高次元 Gaussian を隠れ次元方向に積分すると、低次元にも Gaussian 的な形が残りやすい。  
これは、

\[
\text{高次元状態}\rightarrow\text{見える3D状態}
\]

という投影・周辺化モデルと相性がよい。

### 6.4 時間発展 primitive として扱いやすい

自由粒子や二次ポテンシャルでは、Gaussian 波束は Gaussian の形を保ったまま、中心・幅・位相曲率を変化させる。  
したがって、動的 primitive としても扱いやすい。

## 7. 古典世界への見え方

ミクロでは、観測イベントは確率的に揺れる。

\[
x_{\mathrm{obs}}\sim P(x)
\]

しかし、マクロ世界では以下の要因で揺れが見えにくくなる。

1. 多数回の sample により、ヒストグラムが確率分布へ収束する。
2. 多数の自由度の平均により、相対揺らぎが \(1/\sqrt N\) で小さくなる。
3. 環境との相互作用により、干渉が観測不能になる。これはデコヒーレンスである。
4. 観測装置の有限分解能により、細かい構造が粗視化される。

粗視化は、たとえば次のように書ける。

\[
P_{\mathrm{obs}}(r,t)=\int dr'\,W_\Delta(r-r')P_{\mathrm{vis}}(r',t)
\]

ここで \(W_\Delta\) は観測分解能を表すカーネルである。

したがって、このモデルでは、

\[
\text{complex amplitude cloud}
\rightarrow
\text{probability field}
\rightarrow
\text{sampled events}
\rightarrow
\text{accumulated / coarse-grained classical world}
\]

という流れで、量子的揺らぎと古典的安定性を同じ図式の中で扱う。

- 見えない次元が、低エネルギーでは周辺化・凍結されること
- 観測世界が、高次元状態のレンダー結果として見えること

## 9. まだ閉じていない部分

このモデルは、世界理論としては未完成である。  
特に、次の部分はまだ閉じていない。

### 9.1 splat は実在か、表現か

同じ \(\Psi\) を異なる splat 群で表せる可能性がある。

\[
\sum_j c_jg_j=\sum_k c'_kg'_k
\]

その場合、どちらの splat cloud が本物なのか決まらない。  
したがって現時点では、splat を実在ではなく、状態を表す表現 primitive と見る方が安全である。

### 9.2 Hamiltonian / action が未定

状態表現はあるが、世界がどう時間発展するかを決める本体がまだない。

\[
i\hbar\partial_t\Psi=\hat H\Psi
\]

この \(\hat H\) をどう定めるかが、物理理論としての核心である。

### 9.3 Born rule と一回の結果

このモデルでは、

\[
P=|\Psi|^2
\]

および

\[
x_{\mathrm{obs}}\sim P
\]

を採用している。  
しかし、なぜ絶対値二乗なのか、なぜ一回の観測で一つの結果が実現するのかは、まだ説明していない。

### 9.4 多粒子・もつれ

1粒子なら \(\Psi(x)\) でよい。  
しかし \(N\) 粒子では、

\[
\Psi(x_1,x_2,\dots,x_N)
\]

となり、splat は物理空間ではなく配置空間上に置かれる。  
場の理論まで進むなら、状態は

\[
\Psi[\phi(x)]
\]

のような場の配位空間上の対象になる。

### 9.5 標準模型との接続

現実の物理には、フェルミオン、ボソン、スピン、電荷、色荷、ゲージ対称性がある。

\[
SU(3)\times SU(2)\times U(1)
\]

Gaussian splat 表現だけでは、これらは自動的には出てこない。  
内部状態 \(\chi_j\) や Hamiltonian に、これらの構造を組み込む必要がある。

### 9.6 相対論と重力

現在の書き方は、時間を外部パラメータとして扱う非相対論的形式に近い。  
相対論的に整えるには、Lorentz 対称性、因果律、粒子生成消滅、場の理論が必要になる。

重力まで含めるなら、時空計量

\[
g_{\mu\nu}(x)
\]

そのものを動的変数として扱い、状態は

\[
\Psi[g_{\mu\nu},\phi]
\]

のようになる。ここはまだ大きく開いている。

### 9.7 隠れ次元がなぜプランク長で安定するか

\[
R\sim\ell_P
\]

と置いたが、なぜその値で安定するのかは説明していない。  
本来は、余剰次元のサイズ \(R\) を固定するポテンシャルや機構が必要である。

### 9.8 反証可能性

現時点では、既存量子論を Gaussian splat の語彙で再表現している部分が大きい。  
新しい物理理論として立てるには、既存理論と異なる予測が必要になる。

## 12. 結論

この仮説は、世界を「点粒子の集合」としてではなく、**高次元 complex Gaussian splat cloud の観測レンダー**として見る試みである。

各 splat は、中心・広がり・向き・運動量・位相・複素重み・内部状態を持つ。  
それらは密度としてではなく、複素振幅として重ね合わされる。  
観測者は、隠れ次元を積分消去した確率場を見ており、一回の観測ではそこから一つのイベントが sample される。  
多数回・多数自由度・環境相互作用・粗視化により、ミクロの揺らぎはマクロには安定した世界として現れる。

ただし、これは現段階では実在論ではなく表現論として扱うのがよい。  
未閉鎖の中心は、Hamiltonian / action、Born rule の根拠、標準模型との接続、相対論・重力、隠れ次元の安定化、反証可能性である。

最も安全な言い方は次である。

> 量子状態は、complex Gaussian splat cloud として表現できる。  
> 観測される世界は、その cloud を隠れ次元方向に周辺化し、有限分解能でレンダーし、イベントとして sample した結果である。  
> このモデルは新物理そのものではなく、既存量子論と観測の関係を CG 的に再編成するためのコンパクトな表現フレームである。
