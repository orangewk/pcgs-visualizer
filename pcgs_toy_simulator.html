Complex Gaussian Splat Cloud Toy Simulator

3DGSの語彙に「矢印＝複素位相」を足すと、density render と event render、干渉、デコヒーレンスをどう見られるかを示す最小模型です。

Parameters

λ=1 は完全に干渉する「矢印つき splat」。λ=0 は環境へ位相関係が漏れた後の mixture、つまり古典的な足し算に近い状態です。

ψ₁ = G(x+d/2) exp(+ikx)
ψ₂ = G(x-d/2) exp(-ikx+iΔφ)
P = |ψ₁|² + |ψ₂|² + 2λ Re(ψ₁*ψ₂)
event render: x_obs ~ P(x)

上段: density render = 確率場 P(x)。縦棒は event render のサンプルヒストグラム。下段: 2つの splat の位相矢印。矢印の相対角が揃う場所は明るく、逆向きの場所は暗くなる。
